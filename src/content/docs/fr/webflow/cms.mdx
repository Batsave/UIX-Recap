Le CMS de Webflow permet de gérer du contenu **dynamique** (blog, portfolio, produits) sans écrire une ligne de code.

---

## Collections

- Une **collection** = une base de données interne.
- Chaque collection contient des **champs personnalisés** (texte, image, lien, date, booléen…).
- Les champs peuvent être utilisés dans des **templates de pages CMS**.

---

## Fonctionnement

1. Créer une collection (ex : _Articles_).
2. Ajouter des champs (titre, auteur, image, contenu).
3. Lier les champs aux éléments d’une page (textes, images, boutons).
4. Publier → les pages CMS sont générées automatiquement.

---

## Cas d’usage

- Blog automatisé
- Portfolio dynamique
- Catalogue produit
- Fiches membres d’équipe

---

> [!NOTE]  
> Webflow CMS est pensé pour les **contenus évolutifs** : toute modification dans la base est répercutée automatiquement sur le site.
