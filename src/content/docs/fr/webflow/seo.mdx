Webflow intègre des outils pour améliorer le **référencement naturel (SEO)** sans passer par du code complexe.

---

## Paramètres SEO par page

- **Title Tag** : titre affiché dans Google.  
- **Meta Description** : résumé de la page.  
- **Slug personnalisé** : URL optimisée (mots-clés, tirets).  
- **Balises Open Graph (OG)** : titre, description et image pour le partage sur réseaux sociaux.  

---

## SEO et CMS

- Chaque **Collection CMS** peut inclure ses propres champs SEO.  
- Automatisation possible avec des variables (ex : *Titre de l’article | Nom du site*).  
- Mise à jour centralisée → optimisation rapide d’un grand volume de contenu.

---

## Bonnes pratiques SEO dans Webflow

- Utiliser des **titres hiérarchisés** (H1, H2, H3…).  
- Renseigner les **alt text** des images.  
- Optimiser les **performances** : Webflow gère automatiquement CDN, compression, minification.  
- Créer un **sitemap.xml** et un **robots.txt** directement dans les paramètres du projet.  

---

> [!TIP]  
> Le SEO dans Webflow repose sur une **bonne structure de contenu** : titres clairs, URLs propres et performances optimisées.
