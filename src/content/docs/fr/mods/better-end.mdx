## 4. Progression d’équipement

1. **Thallasium** – Minez *Thallasium Ore* → fondez en *Thallasium Ingot*. Outils/armures via Thallasium Anvil.  
2. **Terminite** – Fondez *Thallasium Ingot* + *End Dust* dans l’End Stone Smelter. Stats > Fer, indispensable pour étapes suivantes.  
3. **Aeternium** – Fondez *Terminite Ingot* + *Netherite Ingot* ; niveau diamant+ avec enchantabilité accrue.  
4. **Crystalite (Infusion)** – Placez plastron/outil Terminite au centre + Amber Gem + Crystal Shards → gear end-game + effets spéciaux.  
5. **Crystalite Elytra / Armored Elytra** – Upgrades d’Elytra offrant protection et durabilité.  

---

## 5. Possibilités de gameplay & construction

- **Esthétique** : palettes bois + pierres variées pour temples, cités volantes, laboratoires futuristes.  
- **Lumière naturelle** : Aurora Crystal, plantes bioluminescentes – évite le spam de lanternes.  
- **Terraforming** : Menger Sponge pour assécher caves sous-marines ; blocs de glace & neige dense pour routes rapides.  
- **Infusion** : créez outils à effets (Vitesse sous l’eau, Feather Falling renforcé, etc.) en combinant gemmes / mobs drops.  
- **Survie longue durée dans l’End** : Respawn Obelisk + Eternal Portals + farms End Fish/Cubozoa = base autonome.
