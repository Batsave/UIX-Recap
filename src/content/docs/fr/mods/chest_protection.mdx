Le mod **Hey That’s Mine (HTM)** vous permet de **protéger vos coffres, fours, tonneaux, etc.** automatiquement sur le serveur.  
Il est conçu pour empêcher tout autre joueur d’accéder à vos conteneurs sans votre autorisation.

> 🔒 Tous les nouveaux coffres sont **protégés automatiquement** dès leur pose.

---

## 🎮 Commandes utiles

Vous trouverez ci-dessous toutes les commandes utiles pour gérer vos protections.  
Une fois la commande tapée, **faites un clic gauche sur le coffre** concerné (sauf indication contraire).

---

### 🔁 Changer le mode de protection

| Commande           | Effet                                                             |
| ------------------ | ----------------------------------------------------------------- |
| `/htm set PRIVATE` | Protège le coffre (par défaut) – vous seul et vos amis y accédez. |
| `/htm set PUBLIC`  | Tout le monde peut l’ouvrir. ❗ À utiliser avec prudence.         |
| `/htm set KEY`     | Seuls ceux avec une **clé spéciale** peuvent y accéder.           |

---

### 🤝 Faire confiance à un joueur

| Commande                       | Effet                                         |
| ------------------------------ | --------------------------------------------- |
| `/htm trust <pseudo>`          | Permet à un joueur d'accéder à **ce coffre**. |
| `/htm trust <pseudo> global`   | Donne accès à **tous vos coffres** au joueur. |
| `/htm untrust <pseudo>`        | Retire l'accès à ce coffre.                   |
| `/htm untrust <pseudo> global` | Retire l'accès à tous vos coffres.            |

---

### 🧹 Retirer la protection

```bash
/htm remove
```

> Supprime toute protection sur un coffre (il devient accessible à tous).

---

### 👑 Transférer un coffre

```bash
/htm transfer <pseudo>
```

> Transfère la propriété du coffre à un autre joueur.

---

### 🔁 Mode persistant

```bash
/htm persist
```

> Permet d’exécuter plusieurs fois une commande sans la retaper à chaque fois (utile en série).

---

### 🔍 Vérifier les infos d’un coffre

```bash
/htm flag
```

> Affiche les infos de protection du coffre ciblé.

---

### ✅ Conseils pratiques

> Utilisez PRIVATE pour vos coffres personnels.

> Utilisez trust global pour donner l’accès à un ami de confiance partout.

> Évitez PUBLIC sauf si c’est volontaire (exemple : zone de dons).

> Utilisez persist quand vous placez plein de coffres d’un coup.
